Installation via curl
This is probably the easiest method of installation... can be installed via the following command or via an equivelant script
  1. First run curl “way-cooler.org/releases/latest/install.sh” -sSf | sh
  2. Assuming this successfully runs, it will download the tarred version of way cooler and run the install script automatically
  3. If issues occur, please refer to troubleshooting below

Via a tarred version of Way-Cooler
This will download Way-Cooler directly, and the install script can be run manually
  1. Download the tarred version, which can be found here
  2. Run the install script
  3. Install the background program **optional
  4. Install the Way-Cooler bar **optional
  5. If issues occur, please refer to troubleshooting below
